# Environment Variables

Backend: `PORT`, `NODE_ENV`, `CLIENT_ORIGIN`, `DATABASE_URL`, `DB_SSL`, `DB_SYNC`, `JWT_SECRET`, `JWT_EXPIRES_IN`, `JWT_REFRESH_SECRET`, `JWT_REFRESH_EXPIRES_IN`, `REFRESH_COOKIE_NAME`, `REDIS_URL`, `CLOUDINARY_CLOUD_NAME`, `CLOUDINARY_API_KEY`, `CLOUDINARY_API_SECRET`, `FIREBASE_SERVER_KEY`, `OPENAI_API_KEY`.

Frontend: `VITE_API_BASE_URL`, `VITE_SOCKET_URL`, Firebase public web config values in `frontend/src/notifications/firebase.js` when push is enabled.
