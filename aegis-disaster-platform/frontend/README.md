# Aegis Frontend

React + Vite frontend for victims, authorities, helpers, NGOs, and operational response teams.

## Folder Purpose

- `components/` - Reusable UI building blocks.
- `pages/` - Route-level screens grouped by feature and dashboard role.
- `layouts/` - Shared shells for public and protected areas.
- `routes/` - Router definitions and protected route guards.
- `redux/` - Redux Toolkit store and feature slices.
- `api/` and `services/` - HTTP client and domain service wrappers.
- `maps/` - Leaflet map components and tracking helpers.
- `sockets/` - Socket.IO client setup and event names.
- `notifications/` - Firebase Cloud Messaging setup.
- `analytics/`, `ai/`, `drone/`, `webrtc/` - Future-ready operational modules.

## Commands

```bash
npm install
npm run dev
npm run build
```
